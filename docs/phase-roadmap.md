# Life's Details Phase Roadmap

## Current Phase

Phase 0 - Product Foundation

## Scope

- Next.js TypeScript application foundation.
- Mobile-first public shell.
- Placeholder public routes.
- Typed local configuration for business information, services, opening hours, and navigation.
- Environment variable validation.
- Basic testing and validation setup.

## Explicitly Out of Scope

- Database-backed booking.
- Customer accounts.
- Admin authentication.
- Payments.
- Reviews, loyalty, referrals, memberships, or automation.

## Next Trigger

Begin Phase 1 when the owner is ready to turn the foundation into a conversion landing page with final service copy, visual proof, FAQs, and analytics events.
